# Development Resources

# Guide

  • Tutorials (opens new window) - Tutorial articles that helps you to explore the VeChain step by step.

  • Awesome list (opens new window) - Here you find more resources from community and helps to get you started.

  • Useful tips for building a dApp (opens new window) - This article is helping you to understand how to use connex2 to build a Dapp and also how it interacts between the user,application, and Sync2.

  • Miscellaneous (opens new window) - a list that helps you to find built-in contract ABIs, chain tag and more.

# Wallets

  • Sync2 (Desktop (opens new window) / Android (opens new window) / build from source (opens new window)) The next-generation VeChain wallet for all platforms.

  • Sync (Desktop (opens new window)) - Provides seamless experience for users and developers. Which is designed to provide the superior user experiences for VeChain Apps, and serves as the dApp environment to provide unlimited potential for developers and users.

  • VeChainThor Wallet (iOS (opens new window) / Android (opens new window)) - a light mobile wallet app to provide users with a secure, simple, fully functional portal of the VeChainThor blockchain.

# Explorers

  • VeChain Explorer (opens new window) - VeChain Explorer enables you to explore and search transactions, addresses, and other activities taking place on the VeChainThor blockchain

  • Insight (opens new window) - A serverless VeChain explorer. It allows you to explore and search for blocks, transactions and accounts.

  • VeChainStats (opens new window) - VeChain Blockchain statistics

# Tools

  • Connex - Connex is the standard interface to connect dApps with VeChain blockchain and users. Aiming to help developers building decentralized applications.

  • web3 providers connex (opens new window) - Web3.js provider implemented using Connex.js. It makes possible to use web3.js to interact with the VeChain Thor protocol.

  • Thor-devkit(JavaScript (opens new window) / Java (opens new window) / Python (opens new window) / C# (opens new window) ) - SDKs for DApp development

  • Thorify (opens new window) - A web3 adaptor for VeChain Thor RESTful API.

  • thor-requests.py (opens new window) - This library enables you to talk to VeChain blockchain without hassle.

  • Rosetta API (opens new window) - Rosetta Blockchain Standard Implementation for VeChain.

  • Online IDE (opens new window) - a browser-based compiler and IDE that enables users to build contracts with Solidity language and to debug transactions.

  • Inspector (opens new window) - A tool that allows you to deploy and interact with the contract.

  • Fee Delegation as a Service (DaaS) (opens new window) - an infrastructure-level module that provides a framework for setting up a fee-delegation service based on the VIP-191 protocol.

  • vechain.energy (Mainnet (opens new window) / Testnet (opens new window)) - Fee Delegation as a Service for VeChain developers

  • Hardhat VeChain Thor plugin (opens new window) - Hardhat-thor allows to deploy and interact with contracts through hardhat with VeChain

# Public Services

  • Testnet faucet (opens new window) - Provides VET and VTHO on testnet freely. It is designed to help developers test applications as well as provide users an alternative to experience dApps on the VeChainThor blockchain

  • App-hub (opens new window) - a platform that allow you to submit your awesome app and discovery by people around the world.

  • B32 (opens new window) - Signature collection of smart contract method & event

  • Connex-env (opens new window) - The landing page leads user to download Connex powered VeChain wallets

  • Token-registry (opens new window) - A platform where you can submit the token.

    • Mainnet token list (opens new window)
    • Testnet token list (opens new window)

  • VeChainThor wallet token registry (opens new window) - Add your VIP-180 or VIP-181 token to VeChainThor wallet

  • Picasso (opens new window) - General purpose deterministic identity icon library in svg format, mostly for VeChain thor addresses.

# Demos

  • Buy me a coffee (opens new window) - A demo allows your supporter to support your work by donating VET

  • Energy Station(Testnet Only) (opens new window) - A demo allows you to exchange energy(VeThor) and VET on VeChain Thor blockchain.

  • My-tokens (opens new window) - A demo allows you to transfer tokens

# Public Nodes

Public nodes allow you to test all the features without setting up your own node. Below are the available nodes for development and testing.

  • Mainnet:

    • https://vethor-node.vechain.com (opens new window)- provided by VeChain Foundation
    • VeBlocks Public Nodes (opens new window)- provided by VeBlocks

  • Testnet:

    • https://vethor-node-test.vechaindev.com (opens new window)- provided by VeChain Foundation
    • VeBlocks Public Nodes (opens new window)- provided by VeBlocks

Note

  1. The public nodes are not designed for the production environment, and VeChain Foundation does not guarantee the reliability and availability.

  2. VeBlocks Public Nodes is a community project to support all developers and allow them everywhere-access to the blockchain, without the need to have a full node with them. For more information, please visit VeBlocks Public Nodes (opens new window)
